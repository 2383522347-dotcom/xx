【清理 Git 合并冲突】

你的 index.html 和 app.js 里还有未解决的合并冲突标记（<<<<<<<、=======、>>>>>>>），会导致页面或脚本报错。

请按下面做一次：

1. 打开文件夹：D:\机电一体化英文学习服务器\xx
2. 在地址栏输入 cmd 回车，打开命令行
3. 输入：node fix-conflict.js
4. 回车执行，看到「完成」后关掉窗口

执行完后，冲突标记会被删除，只保留当前正确版本。然后可以重新用 GitHub Desktop 提交（Commit to main）并推送（Push origin）。

若本机没有安装 Node.js，可以：
- 安装 Node.js：https://nodejs.org/
- 或手动用记事本打开 index.html、app.js，搜索 "<<<<<<<"、"======="、">>>>>>>"，删除从 ======= 到 >>>>>>> 之间的整段重复内容，保存。
