【没有 Git 时】用 GitHub 网页上传「机电一体化英文学习」

1. 在 GitHub 建好空仓库（New repository，不要勾选 README）。

2. 打开该仓库页面，点击 “uploading an existing file” 或 “Add file” → “Upload files”。

3. 从「机电一体化英文学习」文件夹里选中并上传这些（可多选或拖拽）：

   根目录文件：
   - index.html
   - app.js
   - data.js
   - package.json
   - vercel.json
   - run.bat
   - run.command
   - server.js
   - server-sync-example.js
   - GitHub上传与Vercel部署.md
   - README-READING-DATA.md
   - README-SYNC.md
   - 使用说明.txt
   - 无Git上传说明.txt

   文件夹（必须上传）：
   - api 文件夹里的 sync.js（点进 api，选中 sync.js 一起上传）

4. 页面底部 Commit message 填“初版”，点 Commit changes。

5. 然后去 Vercel 里 Import 这个 GitHub 仓库，按《GitHub上传与Vercel部署.md》第三步、第四步做即可。

注意：上传后仓库里要能看到 api/sync.js，否则 Vercel 部署后同步接口会失效。
